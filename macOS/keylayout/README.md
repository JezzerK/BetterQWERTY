# BetterQWERTY (macOS `.keylayout`)

This is the  macOS keyboard layout version of BetterQWERTY.

## Notice:
- It only swaps "P" and ";", not "Caps Lock" and "Backspace  
  
This is because .keylayout files cannot remap caps lock on Mac. 

## Install

1. Download **`betterQWERTY.keylayout`** from this folder on GitHub
2. Move it to: ~/Library/Keyboard Layouts/
3. Log out and log back in
4. System Settings → Keyboard → Input Sources → **+** → **Others**
5. Select **BetterQWERTY**
